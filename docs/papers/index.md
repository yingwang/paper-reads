## 四段结构

每篇笔记按以下顺序展开：

- **问题**：要解决什么、为什么是问题、前人解法在哪里不够
- **方法**：核心做法与关键设计，符号与公式按出现顺序解释清楚
- **实验**：数据集、baseline、主要数字、最具说服力的 ablation
- **局限**：作者自己承认的与读者能看出来的潜在问题

文末有一句 60 字以内的「一句话」小结，给读者未来在 reading list 里快速回忆用。
